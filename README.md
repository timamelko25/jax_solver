# Background

Задачи фильтрации многокомпонентных жидкостей критически важны для повышения нефтеотдачи (EOR), геологического хранения CO₂ и моделирования загрязнения грунтовых вод. В отличие от однофазного случая, здесь ключевую роль играют взаимодействие компонентов, фазовые проницаемости и капиллярные силы, что приводит к системе связанных нелинейных уравнений, где небольшие изменения насыщенности могут кардинально менять картину течения. 

# Task

Реализовать на JAX точный алгоритм решения задачи двухфазного несжимаемого течения (например, вода и нефть) в гетерогенной пористой среде. Система уравнений включает:

1.     Уравнение давления (эллиптическое): , где общая подвижность  зависит от насыщенности

2.       Уравнение транспорта насыщенности (гиперболическое):  (уравнение Бакли-Леверетта)

Цель — создать дифференцируемый солвер на JAX, который корректно разрешает фронт вытеснения (shock wave) при высоких отношениях подвижностей. Особое внимание уделить реализации схем "против потока" (upwind schemes) или использованию PINN с добавлением физических ограничений (entropy conditions) для предотвращения нефизичных осцилляций на фронте насыщенности. 

Related works

·       Библиотека: JAX-LaB — пример реализации многофазных течений на JAX 

·       Датасеты: Stanford AD-GPRS benchmarks или задачи из SPE10 (Society of Petroleum Engineers) — классические тесты для мультифазной фильтрации

·       Статьи: Physics-Informed Neural Networks for Multi-Phase Flow in Porous Media (Buckley-Leverett equation solving)
